LocalVision CMS v2.0.0 Schedule Playlist MVP

핵심 변경사항
1. 업체별 플레이리스트 그룹 기능 추가
   - 기본 플레이리스트 자동 생성
   - 기존 좌측 콘텐츠는 자동으로 기본 플레이리스트에 배정
   - 점심/저녁/주말 등 별도 그룹 생성 가능

2. 시간대별 송출 관리 UI 추가
   - 요일 선택
   - 시작시간 / 종료시간 선택
   - 플레이리스트 그룹 선택
   - ON/OFF 및 삭제
   - 시간 테스트 미리보기
   - 콘텐츠별 플레이리스트 배정

3. API 추가
   - /api/playlist-groups
   - /api/playlist-schedules

4. Player 연동
   - /api/player-state 응답에 playlistGroups, playlistSchedules, activePlaylistGroup 추가
   - 기존 playlists.left/right 응답은 유지해 하위호환 보장

5. APP 수정 없음
   - 기존 APP Shell은 그대로 사용
   - CMS + Player 교체만으로 기능 적용

주의사항
- 기존 D1/R2 데이터 삭제 없이 배포합니다.
- 배포 후 /api/health 또는 CMS 접속 시 ensureCoreSchema가 새 테이블과 컬럼을 보강합니다.
- 스케줄 콘텐츠가 많아질 경우 Player URL cacheMax=50을 권장합니다.
