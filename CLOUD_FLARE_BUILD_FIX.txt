LocalVision CMS v1.6.4 BuildSafe

Cloudflare Pages 배포 실패 원인:
- 기존 package-lock.json 안에 내부 registry URL이 들어가 있어 Cloudflare npm clean-install 단계가 실패할 수 있음.
- 이 BuildSafe 버전은 package-lock.json을 제거하고, package.json 의존성 버전을 고정함.
- .npmrc로 registry를 https://registry.npmjs.org/ 로 고정함.
- .nvmrc/.node-version으로 Node 20 사용을 권장함.

Cloudflare Pages 설정 권장:
- Build command: npm run build
- Build output directory: dist
- Root directory: 비워둠 또는 /
- Environment variable: NODE_VERSION = 20

Bindings:
- D1: DB = localvision-cms-db
- R2: MEDIA = localvision-media-prod
- Text variable: R2_PUBLIC_BASE = R2 public URL
- Optional: ONLINE_TTL_SEC = 600
