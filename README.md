# LocalVision CMS v2.2

## v2.2 핵심

TV 설치용 URL에 Player v1.4 오프라인 번들 캐시 옵션을 자동으로 붙입니다.

## 자동 포함 옵션

- refresh=3600000
- heartbeat=30000
- bundleMode=cache
- cacheAll=1
- videoMode=cache
- activateWhenCached=1
- cacheMax=60
- fit=cover

## 같이 필요한 것

- Player 저장소: LocalVision Player v1.4 업로드
- Android TV App: v2.8 빌드 후 설치
