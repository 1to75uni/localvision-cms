LocalVision CMS v1.7.4 업로드 안내

이 ZIP은 GitHub localvision-cms 저장소를 비우고 그대로 업로드하는 완성본입니다.

업데이트 핵심
- CMS 메인 화면 안에서 업체별 lv001 APP ID 등록/확인/수정 가능
- 왼쪽 메뉴에 APP ID 관리 추가
- 대시보드 현재 선택 업체 카드 아래 APP ID / Player URL 연결 카드 추가
- 업체 관리 화면 상단 APP ID 빠른 확인 추가
- /api/app-config?id=lv001 유지
- /boot.html?id=lv001 유지
- heartbeat=300000, commandPoll=300000, noticePollMs=300000
- ONLINE 기준 10분
- 콘텐츠 기본 재생시간 20초

배포 후 확인
1. /api/health
2. /api/app-config?id=lv001
3. /boot.html?id=lv001
4. CMS 접속 후 왼쪽 메뉴 APP ID 관리 확인
