LocalVision CMS v1.7.1 LV-ID 통합 관리 반영사항

핵심 변경
1. CMS 메인 화면 안에서 업체별 APP ID(lv001, lv002...)를 바로 확인/등록/수정할 수 있도록 통합했습니다.
2. 왼쪽 메뉴에 "APP ID 관리" 버튼이 추가됩니다.
3. 대시보드의 "현재 선택 업체" 카드 아래에 APP ID / Player URL 연결 카드가 표시됩니다.
4. 업체 관리 화면 상단에 APP ID 빠른 확인 영역이 표시됩니다.
5. APP ID 관리 화면에서 다음을 한 번에 처리할 수 있습니다.
   - APP ID 수정
   - Player URL 직접 지정/초기화
   - Player URL 복사
   - TV 설치용 Boot URL 복사
   - /api/app-config?id=lv001 확인
6. 기존 /lv-id-url-manager.html, /boot.html, /api/app-config?id=lv001 구조도 유지됩니다.

운영 기준
- APP은 lv001 같은 업체 ID만 저장합니다.
- CMS는 ID별 최신 Player URL을 반환합니다.
- Player URL 변경은 CMS에서만 처리합니다.
- heartbeat=300000
- commandPoll=300000
- noticePollMs=60000
- ONLINE 기준은 마지막 heartbeat 10분 이내입니다.
- 콘텐츠 기본 재생시간은 20초입니다.
- D1 heartbeat 기록은 10분마다 또는 상태 변경 시만 남기는 방향입니다.

배포 후 확인 주소
- /api/health
- /api/app-config?id=lv001
- /boot.html?id=lv001
- /lv-id-url-manager.html

화면 사용법
1. CMS 접속
2. 왼쪽 메뉴 "APP ID 관리" 클릭
3. 업체별 APP ID가 lv001, lv002 방식으로 보이는지 확인
4. 필요한 경우 APP ID / Player URL 수정 후 저장
5. TV 설치용 Boot URL 복사 후 앱 기본 URL 또는 buly.kr 목적지로 사용
